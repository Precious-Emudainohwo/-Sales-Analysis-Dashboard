## Project Overview
This project is a **Sales Analysis Dashboard** built using Microsoft Excel to analyze retail sales data and provide clear business insights.

The dashboard focuses on understanding:
- Overall sales performance
- Revenue loss from discounts and returns
- Best revenue-generating products
- Most profitable product categories

It is designed to help stakeholders make **data-driven business decisions** quickly.

---

## Dataset Description
The dataset contains retail sales transaction data, including:
- Product categories
- Units sold
- Gross sales
- Discounts
- Sales returns
- Net sales
- Profit and profit margin

Data was cleaned, transformed, and modeled using Excel before visualization.

---

## Dashboard Features

### Key Performance Indicators (KPIs)
The dashboard highlights four main KPIs:
- **Total Revenue**
- **Total Discounts Given**
- **Total Profit Margin**
- **Total Sales Returns**

These metrics provide a quick snapshot of overall business performance.
